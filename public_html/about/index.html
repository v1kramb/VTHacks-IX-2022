About

The purpose of this website is to provide constructive feedback on certain exercises using state of the art computer vision software. The libraries used in this project were from MediaPipe, a Google owned website with ML solutions.

Visit MediaPipe

Pose Functionality

The funciton of MediaPipe's pose software is to offer high-fidelity body pose tracking, predicting the locations of 33 pose landmarks. It accomplishes this by using BlazePose, a software that utilizes machine learning, to track the locations of key body parts and provide pose estimations.

Learn More »

 

Pose Classification

BlazePose is able to classify certain poses as exercises such as pushups and squats through the use of the k-nearest neighbors algorithm to train the model.
